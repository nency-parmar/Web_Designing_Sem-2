Concept Of WWW

  • It Stands For World Wide Web.
  • It is Basically a System of internet serves that supports specially formatted documents.
  • It is collection of text pages, videos, digital photographs and animations we can access over the internet.
  • User access the World Wide Web facilities via cilent called a web browser.
